# Mixed Acid-Base Disorders

## Identification Strategy

1. **Determine the primary disorder** from pH direction
2. **Check compensation** using expected formulas
3. **Calculate anion gap** (even if pH is normal)
4. **Calculate delta-delta ratio** if AG is elevated
5. **Look for discordance** between expected and measured compensation

## Common Mixed Patterns

### Triple Disorders

| Pattern | Example |
|---------|---------|
| AG metabolic acidosis + metabolic alkalosis + respiratory acidosis | Alcoholic ketoacidosis (AG acidosis) + vomiting (met alkalosis) + COPD (resp acidosis) |
| AG metabolic acidosis + metabolic alkalosis + respiratory alkalosis | DKA (AG acidosis) + NG suction (met alkalosis) + sepsis (resp alkalosis) |

### Double Disorders

| Pattern | Clue |
|---------|------|
| Metabolic acidosis + respiratory acidosis | pCO₂ higher than Winter's predicts — respiratory failure + metabolic cause |
| Metabolic acidosis + respiratory alkalosis | pCO₂ lower than Winter's predicts — hyperventilation beyond compensation |
| Metabolic acidosis + metabolic alkalosis | Normal pH with elevated AG — the AG unmasks hidden acidosis |
| Respiratory acidosis + metabolic alkalosis | Common in COPD patients on diuretics |

## Red Flags for Mixed Disorders

- **Normal pH with abnormal AG** → opposing metabolic processes
- **Compensation outside expected range** → additional disorder present
- **Delta-delta > 2** → concurrent metabolic alkalosis
- **Delta-delta < 1** → concurrent non-AG metabolic acidosis
- **AG elevated but pH alkalotic** → metabolic alkalosis + AG acidosis
