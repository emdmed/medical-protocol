# Anion Gap Interpretation

## Calculation

**Anion Gap = Na⁺ − (Cl⁻ + HCO₃⁻)**

- Normal: 8–12 mEq/L (without albumin correction)
- Albumin-corrected: AG + 2.5 × (4.0 − measured albumin g/dL)

## Elevated Anion Gap (> 12 mEq/L) — MUDPILES Mnemonic

| Cause | Key Lab |
|-------|---------|
| **M**ethanol | Osmolal gap elevated |
| **U**remia | BUN/Cr elevated |
| **D**iabetic ketoacidosis | Glucose elevated, ketones positive |
| **P**ropylene glycol | Osmolal gap elevated |
| **I**soniazid / Iron | Drug history |
| **L**actic acidosis | Lactate > 2 mmol/L |
| **E**thylene glycol | Osmolal gap, calcium oxalate crystals |
| **S**alicylates | Salicylate level, mixed AG + respiratory alkalosis |

## Delta-Delta Ratio (Δ/Δ)

**Δ/Δ = (AG − 12) / (24 − HCO₃⁻)**

| Δ/Δ Ratio | Interpretation |
|-----------|----------------|
| < 1.0 | Non-anion-gap metabolic acidosis also present (hyperchloremic) |
| 1.0–2.0 | Pure anion gap metabolic acidosis |
| > 2.0 | Concurrent metabolic alkalosis (HCO₃⁻ higher than expected) |

## When to Calculate

- **Always** check AG when metabolic acidosis is identified
- **Always** calculate delta-delta when AG is elevated
- **Correct for albumin** when available — hypoalbuminemia masks elevated AG
