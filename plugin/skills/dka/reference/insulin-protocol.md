# DKA Insulin Protocol

## Initial Insulin Therapy

| Parameter | Protocol |
|-----------|----------|
| **Bolus** | 0.1 units/kg IV (optional — some centers skip) |
| **Infusion rate** | 0.1 units/kg/hour continuous IV |
| **Alternative** | 0.14 units/kg/hour without bolus |

## Glucose-Based Adjustments

| Glucose Level | Action |
|--------------|--------|
| > 300 mg/dL | Continue current rate |
| 200–300 mg/dL | Reduce to 0.05 units/kg/hour AND start D5 (dextrose 5%) |
| < 200 mg/dL | Reduce to 0.02–0.05 units/kg/hour, continue D5–D10 |

## Target Glucose Reduction Rate

- **Target:** 50–75 mg/dL per hour (2.8–4.2 mmol/L per hour)
- **Too fast (> 100 mg/dL/hr):** Risk of cerebral edema (especially pediatric) — reduce insulin rate
- **Too slow (< 50 mg/dL/hr):** Check insulin infusion, consider increasing rate or bolus

## Critical Rules

- **Never stop insulin** until all resolution criteria are met — even if glucose is normal
- **Always run dextrose** when glucose < 250 mg/dL to allow continued insulin therapy
- **Check potassium before starting insulin** — if K⁺ < 3.3 mEq/L, hold insulin and replace potassium first
