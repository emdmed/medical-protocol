# Critical Vital Sign Values

## Immediate Action Required

| Vital Sign | Critical Low | Critical High |
|-----------|-------------|--------------|
| Systolic BP | < 90 mmHg (hypotension) | > 180 mmHg (hypertensive crisis) |
| Diastolic BP | < 60 mmHg | > 120 mmHg (hypertensive emergency) |
| Heart Rate | < 40 bpm | > 150 bpm |
| Respiratory Rate | < 8 breaths/min | > 30 breaths/min |
| Temperature | < 35.0°C / 95.0°F (hypothermia) | > 40.0°C / 104.0°F (hyperpyrexia) |
| SpO₂ | < 90% (severe hypoxemia) | — |

## Clinical Significance

### Hypotension (Systolic < 90)
- Consider: sepsis, hemorrhage, cardiogenic shock, anaphylaxis
- Immediate: IV fluid bolus, vasopressors if unresponsive

### Hypertensive Emergency (> 180/120 with organ damage)
- Target: reduce MAP by 25% in first hour
- IV antihypertensives (labetalol, nicardipine, nitroprusside)

### Severe Hypoxemia (SpO₂ < 90%)
- Supplemental O₂ immediately
- ABG to confirm PaO₂
- Consider intubation if < 85% despite high-flow O₂

### Fever > 40°C
- Blood cultures before antibiotics
- Active cooling measures
- Consider: infection, drug reaction, malignant hyperthermia
